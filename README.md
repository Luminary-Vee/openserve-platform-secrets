# Openserve Platform Security & Secrets Portal

Enterprise documentation portal for Azure Key Vault, secrets management, security architecture, governance, runbooks, and troubleshooting.

---

## 🚀 Quick Start

### Install MkDocs

```bash
pip install mkdocs mkdocs-material

mkdocs serve
  Open:  http://127.0.0.1:8000/

Build the site
  mkdocs build

Deploy
  mkdocs gh-deploy
When to use each
  mkdocs serve -	local development
  mkdocs build -	production output
  mkdocs gh-deploy -	GitHub Pages deployment
  mkdocs new . -	initial project setup

Documentation:
  https://Luminary-Vee.github.io/openserve-platform-secrets/
